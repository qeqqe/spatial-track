Stereo panner for Linux (spatial audio like).

# Requirements 
- Pipewire
- Opentrack (`.exe` with wine, Outputing `UDP over network` with `NeuralNetwork Tracker`)

# Installation
1. Clone this repository
2. Run with `cargo run` (make sure opentrack is running in background)
